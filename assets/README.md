# Assets Directory

## Icon Files

Place your application icons here:

- `icon.ico` - Windows icon (256x256 or multi-size ICO file)
- `icon.icns` - macOS icon (512x512 or multi-size ICNS file)  
- `icon.png` - Linux icon (256x256 or 512x512 PNG file)

## Quick Icon Creation

1. **Create a 512x512 PNG** with your logo/design
2. **Convert to formats:**
   - Windows: Use https://www.icoconverter.com/ or `magick convert icon.png icon.ico`
   - macOS: Use `iconutil` or https://cloudconvert.com/png-to-icns
   - Linux: Just use the PNG file

## Temporary Solution

The app will build and run without icons, but they improve:
- Professional appearance
- OS integration (taskbar, dock, file manager)
- User experience
